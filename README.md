# ☁️ AWS Cloud Web Server Setup Project

A cloud computing project focused on deploying and managing a web server using Amazon Web Services (AWS). This project demonstrates launching an EC2 instance, connecting securely using SSH and PEM keys, configuring Apache, PHP, MySQL, and hosting a PHP-based website.

---

## 🚀 Features

- Launch AWS EC2 instance
- Secure SSH connection using PEM file
- Linux server configuration
- Apache2 web server setup
- PHP and MySQL installation
- phpMyAdmin configuration
- Database creation and management
- Website deployment on cloud server
- User login and registration system

---

## 🛠️ Technologies Used

- AWS EC2
- Ubuntu Linux
- Apache2
- PHP
- MySQL
- phpMyAdmin
- SSH
- Command Line Interface (CLI)

---

## 📌 Project Workflow

1. Create EC2 instance  
2. Generate and use PEM key  
3. Connect server using SSH  
4. Install Apache, PHP, MySQL  
5. Configure database  
6. Deploy PHP website  
7. Test login & registration system
